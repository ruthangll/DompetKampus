# UTS Dompet Kampus

Aplikasi pencatat keuangan sederhana berbasis React Native dan Expo Go.

## Fitur
- Menambah pemasukan
- Menambah pengeluaran
- Menghitung saldo otomatis
- Menampilkan riwayat transaksi
- Menghapus transaksi

## Teknologi
- React Native
- Expo Go
- JavaScript
- useState
- FlatList
